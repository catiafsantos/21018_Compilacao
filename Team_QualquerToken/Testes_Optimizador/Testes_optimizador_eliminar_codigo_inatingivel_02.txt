/*eliminar_codigo_inatingivel()          # Remove código após saltos sem label

Os exemplos mais comuns são após return, break, continue, goto (quando não há um label imediatamente a seguir que possa ser alvo de outro goto), ou chamadas a funções que nunca retornam  */


EXEMPLO 2


int main() {
    int soma = 0;
    for (int i = 0; i < 10; i++) {
        if (i == 5) {
            printf("Encontrado i = 5, a sair do loop...\n");
            break; // O loop termina AQUI quando i = 5.

            // ---- INÍCIO DO CÓDIGO INATINGÍVEL (nesta iteração) ----
            // Este código só seria alcançável se o break não existisse
            // ou estivesse dentro de outra condição que pudesse ser falsa.
            // Como está logo após um break incondicional DENTRO DO if(i==5),
            // nunca é executado QUANDO i==5.
            // *Nota*: Compiladores mais inteligentes podem até perceber
            // que nada após o break DENTRO DO MESMO BLOCO DE CONTROLO
            // (o bloco do if) pode ser alcançado nessa condição.
            printf("Esta linha dentro do if(i==5) após o break é inatingível.\n");
            // ---- FIM DO CÓDIGO INATINGÍVEL ----
        }
        soma += i;
         printf("i = %d, soma parcial = %d\n", i, soma);
    }
    printf("Soma final: %d\n", soma); // Soma será 0+1+2+3+4 = 10
    return 0;
}
